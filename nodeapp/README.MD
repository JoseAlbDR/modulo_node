# NodeApp

Website and API application

## Install

Install dependencies:

```sh
$ npm install
```

Review database connection on /lib/connectMongoos.js

Load initial data:

```sh
  # this command deltes all the data in the database and create default data
  $ npm run init-db
```

## Start

In production:

```sh
  npm start
```

In development:

```sh
  npm run dev
```
